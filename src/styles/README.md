# `src/styles`

Глобальные слои стилей проекта.

Состав:
- `foundation.css` — дизайн-токены (цвета, типографика, сетка, spacing, layout rules);
- `ui-kit.css` — общие primitives для UI-ассетов (например `ui-icon-mask`).

Правила:
- новые токены добавляются в `foundation.css`;
- кросс-страничные утилиты и primitives добавляются в `ui-kit.css`;
- стили конкретной страницы остаются в ее локальном файле (например `App.css`).
